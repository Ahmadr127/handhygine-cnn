Gambar 4.Y Deployment Diagram Sistem Monitoring Kepatuhan Cuci Tangan

Penjelasan: Deployment diagram ini menunjukkan topologi perangkat keras dan perangkat lunak yang saling terhubung. Server utama memuat aplikasi web (Laravel, PHP, Apache), database (MySQL), dan servis kecerdasan buatan (Python, FastAPI, YOLOv8). Node Kamera bertugas merekam dan mengirimkan aliran video (video stream) ke server. Pengguna (Admin/Petugas PPI) mengakses sistem melalui Web Browser pada perangkat PC atau Smartphone menggunakan jaringan lokal maupun internet.
